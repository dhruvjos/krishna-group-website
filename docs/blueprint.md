# **App Name**: Krishna Group Website

## Core Features:

- Minimalist Header: Brand logo on the left, login toggle on the right.
- Course Menu: Display available courses in an option menu.
- Content Sliders: Sliders showcasing content ads and promotions.
- Course Information: Information about camps and specific courses.
- Comprehensive Course List: Detailed listing of courses: English speaking, Leadership, Interview prep, Corporate skills, Aarya maths, Languages (German, Japanese, French), Antivirus, Memory management, Personality development, IELTS, TOEFL, 4s meditation.
- Social Media Footer: Footer with links to Facebook, Instagram, YouTube, email, and contact information.
- Backend System: Robust backend system for content management, user authentication, and course administration. This includes database management, user roles, and analytics tracking.
- WhatsApp Registration Acknowledgment: Send an acknowledgment message to the user's mobile number on WhatsApp after registration.

## Style Guidelines:

- Primary color: Based on the Krishna Group logo's dominant blues and oranges.
- Secondary color: Light gradient background inspired by the logo.
- Headline font: Bold sans-serif, easy to read.
- Body font: Clean, readable sans-serif for descriptions.
- Simple, professional icons for menu items and social media links.
- Clean, grid-based layout to ensure content is organized.
- Responsive design for optimal viewing on all devices.
- Unique and special, yet smooth and eye-pleasing animation effects throughout the website.
